# User module

Both male / female user can set their own distinguishable traits. These traits can benefit matching rate.

## Male traits

The following traits are all optional:

- Nationality
- Region
- Age
- Height
- Weight
- habbits
- description

## Female traits

The following traits are all optional:

- Nationality
- Region
- Age
- Height
- Weight
- Cup
- Habbits
- description
